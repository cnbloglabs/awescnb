## 特性

- 🎨 **丰富的皮肤库** - 提供多个精心设计的皮肤，支持自定义配置
- 🔌 **强大的插件系统** - 内置 30+ 实用插件，轻松扩展皮肤功能
- 📦 **模块化架构** - 核心功能封装为独立包，便于复用和维护
- 🛠️ **完善的开发工具** - 基于 Vite 的快速开发体验，支持热更新
- 🎯 **TypeScript 支持** - 完整的类型定义，提供更好的开发体验
- 📱 **响应式设计** - 所有皮肤均支持桌面、平板和移动设备
- 🌗 **深色模式** - 内置深色/浅色模式切换支持
- ⚡ **性能优化** - 代码分割、懒加载等优化策略
